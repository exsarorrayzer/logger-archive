# Logger Archive

A growing collection of ready-to-use logging and data collection scripts, organized by delivery method and use case. This repository is meant to be a reference archive — structured, clean, and easy to pick up.

> This is just the beginning. More modules and categories will be added over time.
---

## Usage

Each subdirectory contains its own `README.md` with setup and configuration instructions.

---

## Notes

- All scripts are standalone and dependency-minimal by design.
- The archive is organized so modules can be picked individually without pulling the entire repo.
- Contributions and additions are welcome as the archive grows.
